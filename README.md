# FlightExpert Automation Cucumber

## 🚀 Overview
This project is a **Cucumber-based automation framework** using **Selenium and TestNG** for web testing.

## 🛠️ Tech Stack
- **Cucumber** (v7.2.3)
- **Selenium WebDriver** (v4.17.0)
- **TestNG** (v7.8.0)
- **Allure Reporting** (v2.24.0)
